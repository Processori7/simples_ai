# Lab 3 — Свёрточная нейронная сеть (CNN) для классификации изображений

## Описание

Лабораторная работа по обучению свёрточной нейронной сети (Convolutional Neural Network) для классификации изображений на два класса: **медведь** и **бинокль**.

В отличие от Lab 2 (FFNN), здесь используется CNN с аугментацией данных, что позволяет лучше извлекать пространственные признаки из изображений.

## Структура проекта

```
lab3/
├── NS_CNN_1.py               # Обучение CNN модели
├── classifier.py             # GUI-приложение для классификации
├── bear_bin_classifier.h5    # Сохранённая модель CNN
└── 256_ObjectCategories/     # Датасет (не изменять)
    ├── 009.bear/
    └── 012.binoculars/
```

## Компоненты

### 1. NS_CNN_1.py — Обучение модели

Скрипт загружает изображения, применяет аугментацию данных и обучает CNN модель.

**Архитектура модели:**

```
Conv2D(32, 3x3, relu) → MaxPooling2D(2x2)
Conv2D(64, 3x3, relu) → MaxPooling2D(2x2)
Conv2D(128, 3x3, relu) → MaxPooling2D(2x2)
Flatten → Dense(128, relu) → Dropout(0.5) → Dense(2, softmax)
```

**Параметры:**
- Размер изображения: 100x100 пикселей
- Оптимизатор: Adam
- Функция потерь: categorical_crossentropy
- Эпохи: 100
- Размер батча: 32
- Разделение: 80% train / 20% test

**Аугментация данных:**
- Случайное вращение (±20°)
- Сдвиг по ширине и высоте (±20%)
- Shear transformation (±20%)
- Масштабирование (±20%)
- Горизонтальное отражение

**Процесс:**
1. Загрузка изображений из `012.binoculars` (метка 0) и `009.bear` (метка 1)
2. Ресайз до 100x100, нормализация пикселей [0, 1]
3. Разделение на train/test
4. One-hot кодирование меток
5. Создание генератора аугментации
6. Обучение модели с аугментацией
7. Оценка на тестовой выборке
8. Сохранение модели в `bear_bin_classifier.h5`
9. Визуализация графиков точности и потерь

### 2. classifier.py — GUI-приложение

Интерфейс на tkinter для тестирования обеих моделей (FFNN и CNN):

- Выбор модели из выпадающего списка
- Загрузка и предпросмотр изображения
- Классификация с визуализацией вероятностей
- Поддержка инвертирования меток для совместимости моделей
- Цветовая индикация уверенности
