# Lab 2 — Полносвязная нейронная сеть (FFNN) для классификации изображений

## Описание

Лабораторная работа по обучению полносвязной нейронной сети (Feed-Forward Neural Network) для классификации изображений на два класса: **медведь** и **бинокль**.

Используется датасет Caltech-256 (категории `009.bear` и `012.binoculars`).

## Структура проекта

```
lab2/
├── NS_PR_1_ffnn.py           # Обучение FFNN модели
├── classifier.py             # GUI-приложение для классификации
├── bear_bin_ffnn.keras       # Сохранённая модель FFNN
├── ffn.png                   # Графики обучения
└── 256_ObjectCategories/     # Датасет (не изменять)
    ├── 009.bear/
    └── 012.binoculars/
```

## Компоненты

### 1. NS_PR_1_ffnn.py — Обучение модели

Скрипт загружает изображения из датасета, подготавливает данные и обучает FFNN модель.

**Архитектура модели:**

```
Dense(512, relu) → Dropout(0.4) → Dense(256, relu) → Dropout(0.3) → Dense(128, relu) → Dense(2, softmax)
```

**Параметры:**
- Размер изображения: 100x100 пикселей
- Входной вектор: 30000 элементов (100 × 100 × 3)
- Оптимизатор: Adam
- Функция потерь: categorical_crossentropy
- Эпохи: 100
- Размер батча: 32
- Разделение: 80% train / 20% test

**Процесс:**
1. Загрузка изображений из папок `009.bear` (метка 1) и `012.binoculars` (метка 0)
2. Ресайз до 100x100, нормализация пикселей [0, 1]
3. Flatten — преобразование в плоский вектор
4. One-hot кодирование меток
5. Обучение модели
6. Оценка на тестовой выборке
7. Сохранение модели в `bear_bin_ffnn.keras`
8. Визуализация графиков точности и потерь
